Hybrid Search Engine (Lexical + Semantic Fusion)

This project demonstrates how modern AI search systems combine lexical search (TF-IDF) and semantic similarity to produce more robust ranking results.

Architecture

User Query
→ Lexical Similarity (TF-IDF)
→ Semantic Similarity (Vector Space)
→ Score Normalization
→ Weighted Fusion
→ Ranked Results

Why Hybrid Search?

Lexical search captures exact keywords

Semantic search captures meaning

Hybrid fusion combines both strengths

Run Locally
pip install scikit-learn numpy
python hybrid_search.py
